First we need to derive the symbolic jacobian matrix from the urdf with the help of MATLAB/Simulink Robotic toolbox.
We load the urdf into simulink using smimport command, then we can see a tree plot and preview of our robot:

Now we can easily find the geometric parameters and inertial properties of our robot like the transformation matrix, inertia tensor and mass. We can see there are 4 links and 4 joints in a single leg, 3 actuated revolute joint and 1 welded joint. To calculate the jacobian matrix of the end effector we can just simply multiply all the translation and rotation matrix to get the end effector position the take a derivative of joint angles. To do that we take advantage of the MATLAB symbolic toolbox.
